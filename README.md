DönersMan, a product‑level mobile application that started as a random idea during a course and grew into a fully functional, ready‑to‑publish restaurant app. designed around real user flows, persistence, and scalable architecture, not just a UI demo

- Core Features
• Category-based menu with search & favorites
• Persistent cart with full session recovery
• Multiple saved delivery addresses + use current location (GPS)
• Loyalty & reviews for customer engagement
• One-tap contact actions (call, email, open in maps – Al-Sa’a Roundabout)
• Fully consistent light/dark theme

#DonersMan is built with Expo-cli.

Following are the steps for running it for the first time,
"Splash screen" or "Loading screen" can be changed from app.json file, as well as the name of the project.

How to run:
1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```
